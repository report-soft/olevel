Kasagama S.S Report Card Management System (KRCMS)






How to Generate Report Cards

Export to Excel: Click the settings Icon. Choose the "Export to Excel" option to create an empty excel template. You will fill this with learner results.

Upload Badge: Click the Settings Icon. Choose the "Upload Badge" option to add your school badge. It will appear on the report card.

Generate Reports: Click the Settings Icon. Choose the "Generate Reports" option to print out students report cards. A pdf file is generated with all students report cards.

Delete All Records: Click the Settings Icon. Choose the "Delete All Records" option to Delete all records for the current class before importing another class from excel.
